# TSLint

> **An extensible linter for the TypeScript language**

The [`@berun/runner-tslint`](https://www.npmjs.com/package/@berun/runner-tslint) package provides a high level fluent interface for configuring and using [`TSLint`](https://tslint.org).

## Installation

[`@berun/runner-tslint`](https://www.npmjs.com/package/@berun/runner-tslint) is a common runner that is automatically
included  by BeRun starter templates such as [`@berun/template-react`](https://www.npmjs.com/package/@berun/template-react).   If you 
need to install it directly, follow these directions:

```bash
$ npm install --save-dev @berun/runner-tslint
```

```js
// berun.config.ts
export default {
  use: [ '@berun/runner-tslint']
}
```

## Example usage

```js
  berun.tslint.defaultSeverity('warning')
```
