# React

> **React preset using Webpack, modeled after `create-react-app`**

This preset creates a configuration that, unmodified, is identical to `create-react-app`.  You may
add or subtract to it to configure it exactly as needed.

## Installation

```bash
$ npm install --save-dev @berun/preset-react
```

```js
// berun.config.ts
export default {
  use: [ '@berun/preset-react']
}
```

## Included build tools

* Babel v7+
* Webpack v4+
* Jest 
* Typescript (if needed)

## Sparky tasks

```bash
# Run development server on localhost:3000
npm start

# Production build
npm run build
```
